### Installation

This portfolio requires [Node.js](https://nodejs.org/) v10.15.0+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd sandrotchikovani-ssr-app
$ yarn install
$ yarn dev
```

For production environments...

```sh
$ yarn start
```
